Experiences

Internship: Web Developer on X3DPhoto Application

March - August 2017
Dassault Systèmes, Paris

X3DPhoto is a web application developed by X3DPhoto team in CATIA Web Application department. This application allows the users to perform the 3D modeling process, such as 3D reconstruction from multiple images, directly from browser.

During the internship, my main mission is to implement the application tool for painting/texturing 3D object. In this context, my works go with specifying the tool components, creating the user interface and testing the functionality of the tool in the application.

Technos: HTML5, CSS3, JavaScript, ThreeJS, WebGL, Karma-Jasmine, etc.

Internship: Research Engineer

February - August 2016
Laboratoire de Traitement et Transport de l'Information, University of Paris 13

This is the internship at the end of my BSc program. It is in the image processing field and the topic is Image denoising using the theory of Sparse Learning Dictionary. My works go with studying some state of the art method in Learning Dictionary such as K-SVD, MOD, etc, developing an algorithm based on these methods and testing the performance of this algorithm.

Environment: Matlab, C/C++.

Projects

Below is list of some projects developed during my study or in my free time:

Search Music On Spotify - This project is an web application implemented the Spotify API. It fetches the data from Spotify server according to the search input being artist name or track name. Using this application, you can find all albums of artist, or list all songs with given track name, then listen to the preview or explore them on Spotify website.
I built this application by using HTML5, CSS3, Nodejs, Angular4 and Heroku for web hosting service. Souce code can be found on GitHub .
Digi Library - The objective of this project is to build a web application which provides the functionality for management of a local library, e.g., show what books are available, allow new books to be created, update the status of the copies (available, maintenance), etc.
The project is built using HTML5, CSS3, Nodejs, MongoDB, Pugjs and is hosted by Heroku PaaS. Source code can be found on GitHub.
Météo Application - This project is built using the API provided by openweathermap.com to look for weather information of given location and display them by using HTML and CSS. Source code can be found GitHub.
